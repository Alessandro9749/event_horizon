# Security Policy

## Supported Versions

EventHorizon is a script-based project.
Only the latest version available on the `main` branch is supported.

Older versions may contain unpatched issues and are not actively maintained.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

- Do **not** open a public issue for security problems
- Do **not** share exploit details publicly

### How to report

You can report security issues by:
- Opening a **private GitHub security advisory**, or
- Contacting the maintainer directly via GitHub

Please include:
- A clear description of the issue
- Steps to reproduce (if possible)
- Potential impact

## Response

Security reports will be reviewed as soon as possible.
If a valid issue is confirmed, a fix will be released and documented.

Thank you for helping keep EventHorizon safe.
